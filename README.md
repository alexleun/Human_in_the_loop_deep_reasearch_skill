# Human Guided Deep Research Skill

A dedicated skill folder designed to integrate seamlessly into the `skills` directory of **OpenCode** and **OpenSpec** environments. This skill enables AI agents to conduct deep, iterative web research guided by real-time human feedback and course-corrections.

## 📁 Repository Structure
This repository contains only the standalone skill package and its associated assets:
* `deep-research/` - The core skill logic, workflow configurations, and state management files to be placed inside your OpenCode skills directory.

---

## 🛠️ Purpose & Motivation
This project is created and maintained by an **independent developer**. 
* **Built for Learning:** This repository is purely a personal project born out of interest to experiment with, explore, and master the capabilities of open-tooling frameworks like OpenCode and OpenSpec.
* **Modular Design:** It is kept minimal and focused strictly on the skill files to maintain clean modularity within your open agent setups.
